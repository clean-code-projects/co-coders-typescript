# co-coders TypeScript repository

This repository contains TypeScript implementation of `co-coders`.

## Contribute

```sh
# install
yarn

# run tests
yarn test

# run tests in watch mode
yarn watch

# run coverage
yarn coverage

# run build
yarn build

# run lint
yarn lint

# run dependency check
yarn depcheck

# add dev-dependency
yarn add -DW <package>
```
